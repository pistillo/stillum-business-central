=== NEXT STEPS ===

1. Copy environment file:
   cp .env.example .env

2. Start infrastructure:
   docker-compose up -d

3. Build backend services:
   for svc in registry-api publisher runtime-gateway; do
     cd $svc && mvn clean package && cd ..
   done

4. Build frontend:
   cd portal-ui && npm install && npm run build && cd ..

5. Start development services:
   # Terminal 1: Registry API
   cd registry-api && mvn quarkus:dev
   
   # Terminal 2: Publisher
   cd publisher && mvn quarkus:dev
   
   # Terminal 3: Runtime Gateway
   cd runtime-gateway && mvn quarkus:dev
   
   # Terminal 4: Portal UI
   cd portal-ui && npm run dev

6. Access services:
   - Portal UI: http://localhost:3000
   - Registry API: http://localhost:8082
   - Publisher: http://localhost:8083
   - Runtime Gateway: http://localhost:8081
   - Keycloak: http://localhost:8080
   - Temporal UI: http://localhost:8088
   - MinIO: http://localhost:9001
